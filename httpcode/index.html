# 响应码格式

仅供参考，具体值待商量确定

状态码 提示文字
900 正常返回数据
非900 场景异常待具体确定
400 请求错误
401 token值无效，请重新登录
403 拒绝访问
404 请求错误，未找到该资源
405 请求方法未允许
408 请求超时
500 服务器端出错
501 网络未实现
502 网络错误
503 服务不可用
504 网络超时
default message
